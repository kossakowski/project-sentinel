# Project Sentinel

Real-time monitoring bot that scans PL/EN/UA/RU media sources for military attacks against Poland and the Baltic states, classifies via Claude Haiku 4.5, and alerts via Twilio phone call.

**Status:** running in production on a Hetzner VPS.
